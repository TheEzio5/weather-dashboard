# 🌤 Weather Dashboard

Weather Dashboard is a command-line application written in Python that retrieves real-time weather information from an online weather API.

This project focuses on learning how to work with REST APIs, HTTP requests, JSON data, and error handling while building a practical weather application.

This is my fifth portfolio project and my first application that communicates with an external web service.

---

## ✨ Planned Features

- Search weather by city
- Current weather conditions
- Temperature
- Humidity
- Wind speed
- Weather description
- Error handling
- Input validation
